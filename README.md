# AI Image Generator 🎨

This is a simple AI Image Generator web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Generate images based on user input
- Uses dynamic image APIs
- Responsive UI
- Handles async operations in JavaScript

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 💡 Learnings
- DOM Manipulation
- Async/Await
- API Handling
- Debugging (CORS issues)
